# **第 23 堂：華語流行 (Mandopop) 實戰：人聲處理與去齒音**

華語流行音樂（Mandopop）在全球華人世界中擁有極大的影響力，從鄧麗君的柔情時代到周杰倫的 R&B 革新，再到今日多元融合的音樂風格，Mandopop 的人聲處理始終是製作中最核心的一環。與西方流行音樂相比，Mandopop 的人聲往往更強調親密感（intimacy）、清晰度（clarity）與情感傳遞（emotional delivery）。這堂課將從零開始，帶領你走過一條完整的 Mandopop 人聲處理鏈，從原始錄音到最終成品，逐一拆解每一個環節的實戰技巧。

---

## Mandopop 人聲的聲音特質

在動手處理之前，我們必須先理解 Mandopop 人聲的「聲音目標」。相較於西方流行樂經常將人聲融入整體編曲之中，Mandopop 的傳統更傾向於讓歌手「站在最前面」——人聲不僅是樂器之一，更是整首歌的情感核心。

**Mandopop 人聲的三大核心特質：**

1. **近距離感（Close / Intimate）**：人聲聽起來就像歌手在你耳邊傾訴。這需要靠近距離收音（約 10-15 公分）、良好的防噴罩，以及後製中保留適當的中低頻溫暖感。
2. **極高清晰度（Crystal Clear）**：每一個字、每一句的咬字都必須清清楚楚。這意味著中高頻（2-5 kHz）需要有足夠的存在感，同時又要避免刺耳的齒音。
3. **情感動態（Emotional Dynamic）**：主歌輕柔、副歌激昂的對比是 Mandopop 的靈魂。壓縮器不能把動態扼殺，而是要「雕塑」動態，讓情感層次更加分明。

實務上，一個典型的 Mandopop 人聲軌道會包含：主唱（Lead Vocal）、和聲（Harmony / BV）、雙軌加寬（Double Track），以及 ad-lib 或即興點綴。每一層都有不同的處理策略，但整體必須融合成一個立體且有凝聚力的聲音畫面。

---

## 建立人聲處理鏈（Vocal Chain）

人聲處理鏈是訊號依序通過的效果器順序。雖然沒有絕對正確的順序，但 Mandopop 實戰中有一條非常成熟的標準鏈路：

**EQ（等化器）→ Compressor（壓縮器）→ De-esser（去齒音）→ Reverb（殘響）→ Delay（延遲）**

這條鏈路的邏輯是：先用 EQ 清理不必要的頻率，再用壓縮器控制動態範圍，然後針對壓縮後可能被放大的齒音進行去齒音處理，最後在乾淨的訊號上加入空間感效果（Reverb 和 Delay）。當然，實務上你可能會使用多個 EQ 和壓縮器串聯，但這個基本順序是很好的起點。

在數位錄音工作站（DAW）中，通常會將這些效果器以 insert 方式掛在主唱軌道上，而 Reverb 和 Delay 則常用 send/return（輔助軌）方式，以便對多個軌道共用同一個空間效果。

---

## EQ：為 Mandopop 人聲定調

EQ 是人聲處理的第一步，也是最關鍵的一步。以下是針對 Mandopop 主唱的一套標準 EQ 策略：

**高通濾波（High-Pass Filter）**：從 80 Hz 左右開始滾降（slope 12-24 dB/oct）。這可以去除麥克風近距離收音時產生的低頻轟鳴聲（proximity effect）、地板震動聲以及呼吸低頻。請注意不要切太高，否則人聲會失去溫暖的基調。男性歌手可保守一點設在 60-80 Hz，女性歌手則可設在 80-100 Hz。

**低頻處理（100-300 Hz）**：這個區域決定了人聲的「溫暖度」與「厚實感」。如果歌手聲音偏薄，可在 150-200 Hz 做 2-3 dB 的寬 Q 值提升；如果聽起來模糊或鼻音過重，則在此範圍做 2-4 dB 的衰減。

**中低頻（300-800 Hz）**：這是容易產生「盒狀感」或「悶感」的區域。Mandopop 追求清晰度，通常會在此區域做 1-3 dB 的輕微衰減，頻點可選在 400-600 Hz 之間，具體取決於歌手的音色。

**存在感頻段（3-5 kHz）**：這是 Mandopop 人聲的「魔法頻段」。在此區域提升 2-4 dB 可以讓人聲更突出、更有穿透力，在手機喇叭或耳機上尤其明顯。建議使用寬 Q 值（0.5-0.7）的鐘型濾波器，並用 Solo 反覆聆聽，確保聽起來「明亮」但不「刺耳」。

**空氣感（8-12 kHz）**：在此區域做 1-3 dB 的輕微提升（使用 shelf 濾波器），可以增加人聲的空氣感和高頻延伸，讓聲音聽起來更 modern、更精緻。如果齒音問題嚴重，則需謹慎使用此區域，或搭配後續的 de-esser 處理。

實戰提示：在混音過程中，請勿對單一曲目做超過 ±6 dB 的 EQ 調整。大幅度的 EQ 往往暗示收音環節出了問題，應回頭改善錄音品質，而非用 EQ 強行修補。

## 壓縮器：雕塑動態

壓縮器在 Mandopop 人聲中的作用不是把聲音壓扁，而是讓歌手的動態範圍更可控，使低聲呢喃和高聲吶喊之間的聲音落差不會過大，同時保留情感的自然起伏。

**推薦起始設定：**

- **Ratio（壓縮比）**：2:1 到 3:1。Mandopop 不需要像搖滾樂那樣的重壓縮，溫和的比率可以保留情感的動態細節。
- **Threshold（閥值）**：設定在歌手的平均音量之下約 4-6 dB，讓壓縮器在大多數時間都有 2-6 dB 的增益衰減（Gain Reduction）。
- **Attack（啟動時間）**：10-30 ms（慢啟動）。慢啟動讓聲音的初始瞬態（transient，如字頭的爆發力）得以穿透，保留咬字的清晰度。
- **Release（釋放時間）**：40-80 ms（中快釋放）。讓壓縮器在下一個字開始前恢復，避免 pumping 效應。

**增益補償（Make-up Gain）**：壓縮器會降低整體音量，因此需要補償增益，讓處理前後的音量聽起來一致。一般補償 3-6 dB。

許多 Mandopop 混音師會使用串聯壓縮策略（serial compression）：先用一個壓縮器做 2-3 dB 的輕壓（例如 LA-2A 風格的光學壓縮器），再用第二個壓縮器做進一步的動態控制（例如 SSL 風格的 VCA 壓縮器）。這種做法的好處是每個壓縮器的工作量較小，聲音更自然。

**進階技巧**：在副歌段使用更快的 Attack（5-10 ms）和更低的 Threshold，讓副歌人聲被壓縮得更多，聽起來更「貼臉」；主歌段則放寬壓縮參數，保留更多的動態和呼吸感。這可以透過 automation 或動態處理器的側鏈（sidechain）來達成。

## 去齒音（De-essing）：Mandopop 的關鍵戰役

### 什麼是齒音（Sibilance）？

齒音是指人聲中「s」、「sh」、「ch」、「z」、「ts」等發音時產生的高頻能量，通常在 5-10 kHz 之間。這些聲音在錄音中容易被麥克風的高頻響應放大，經過壓縮器之後更是雪上加霜——因為壓縮器會提高整體音量，同時也把齒音跟著放大。

### 為什麼 Mandopop 需要特別注意去齒音？

Mandopop 的人聲特色——近距離收音、高清晰度、頻繁的壓縮——恰好都是齒音的「幫兇」。加上中文語言的聲調特性與大量的齒擦音（sibilant consonants），使得 Mandopop 比起許多西方語種更需要精準的去齒音處理。一個沒有處理好的齒音，會讓聽眾在耳機中感到刺耳、疲勞，尤其是長時間聆聽時。

### 寬頻去齒音 vs. 分頻去齒音

**寬頻去齒音（Wideband De-essing）**：這是最傳統的方法。當偵測到齒音頻段的能量過高時，整個訊號的音量會被短暫壓低。這種方法的優點是簡單、即時，缺點是每當有齒音出現時，整體人聲聽起來會「抖動」一下，不夠平滑。

**分頻去齒音（Split-band De-essing）**：這是現代混音的主流做法。僅針對齒音的頻段（通常是 5-10 kHz）進行壓縮，其他頻段不受影響。這樣做的好處是：當處理齒音時，人聲的主體（中低頻）完全不受影響，聽起來非常自然。

### 實戰操作步驟：

1. **找到齒音頻段**：用 EQ 的 solo 掃描 4-10 kHz 範圍，找到歌手齒音最明顯的頻點。每位歌手、每首歌都不一樣，女性歌手的齒音通常比男性高 1-2 kHz。
2. **設定閾值**：讓 de-esser 在齒音出現時有 3-6 dB 的衰減量。過度的去齒音會讓人聲聽起來「悶掉」或失去高頻細節。一個好的原則是：戴上耳機聽整首歌，只有在專注聽齒音時才感覺到它在作用，而正常聆聽時不應察覺去齒音的存在。
3. **偵測模式**：有些 de-esser 可以選擇偵測頻段（detection band）與處理頻段（processing band）分開。通常偵測頻段設定在 7-9 kHz，處理頻段則涵蓋 4-10 kHz，這樣可以更精準地只攻擊齒音而不傷及人聲。

**進階技巧**：在混音的最後階段，將整首歌用一對好的監聽耳機以較大音量聽過一遍，用鉛筆記錄下「哪一秒出現刺耳的齒音」，然後在該處插入 automation 進行手動處理。這雖然費時，卻是最精準、最自然的去齒音方式。

## 創造人聲深度：殘響與延遲

Mandopop 的人聲需要「近距離感」，但同時也需要適當的空間感來讓聽覺不那麼乾澀。關鍵在於：大量使用輔助軌（send）而非插入軌（insert），讓乾訊號（dry signal）保持在正中間不動，再從輔助軌加入濕訊號（wet signal）。

### 短 Plate 殘響（Short Plate Reverb）

Plate reverb 是 Mandopop 人聲的標準配備。它比 Hall reverb 更緊湊、更集中，不會吃掉人聲的清晰度。

- **Decay（衰減時間）**：1.0 - 1.8 秒
- **Pre-delay（預延遲）**：20-40 ms（讓殘響在人聲發出後才進來，保持咬字清晰）
- **Mix**：10-20%（非常少量！殘響只是要讓聲音有「空氣感」，而非明顯的回音效果）

### Slap Delay（拍擊延遲）

Slap delay 是另一個 Mandopop 的秘密武器。它是在殘響之前加入的短延遲，模擬早期反射聲（early reflection），可以讓人聲聽起來更「立體」而無需添加太多殘響。

- **Time**：60-120 ms（取決於歌曲速度）
- **Feedback**：1-2 次重複
- **Mix**：5-15%（同樣要保持幾乎不可察覺的程度）

實戰中，將 Plate Reverb 和 Slap Delay 分別掛在不同的輔助軌上，各自送出約 -10 到 -15 dB 的訊號到主唱軌道。如果你發現關掉這些效果後人聲變得「很乾」，那就代表量剛剛好；如果有明顯感覺到殘響或延遲的存在，那就是加太多了。

## 雙軌錄音與人聲拓寬

### 雙軌錄音（Double Tracking）

許多 Mandopop 歌曲會在副歌或重點段落使用雙軌錄音。歌手將同一句歌詞唱兩次（甚至三次），分別位於左右聲道，這樣會產生極其自然的加寬效果。實務上：

- 主唱完全置中（Center）
- 第一雙軌（Double 1）偏左 30-40%
- 第二雙軌（Double 2）偏右 30-40%
- 雙軌的音量比主唱低 6-10 dB

如果沒有條件請歌手再唱一次，可以使用 Doubler 效果器或微調音高（detune ±5-8 cents）配合些微延遲（10-20 ms）來模擬雙軌效果，但效果遠不如真實錄音。

### 立體聲加寬工具

對於雙軌之外的單聲道素材（如主歌段的單軌主唱），可以使用立體聲加寬工具（如 iZotope Ozone Imager、Waves S1）來增加寬度。關鍵是**只加寬中高頻（3 kHz 以上）**，低頻維持置中，否則人聲會失去凝聚力。

## 和聲編排（Background Vocal Arrangement）

Mandopop 的和聲編排有幾種常見模式：

**三度和聲**：最傳統的華語流行和聲，在主旋律下方三度（或上方三度）疊加一個和聲軌道。注意中文音調與和弦的配合，避免產生不和諧的衝突音。

**音階堆疊（Stack）**：將一個句子錄製 3-5 次相同旋律，左右平均分布（L100, R100, L80, R80, Center），形成「合唱感」。常見於副歌的後半段，製造推向高潮的效果。

**八度重疊**：男歌手的低八度和聲襯托在主唱下方，可以增加副歌的厚重感。反之，女歌手的高八度疊加則可以增加亮度與穿透力。

**和聲的 EQ 與壓縮**：和聲通常要比主唱做更多的削減——高通濾波切到 200-300 Hz，減少不必要的低頻；高頻也可適度衰減，避免和聲搶了主唱的鋒頭。壓縮比可以比主唱更大（3:1 到 4:1），讓和聲維持在穩定的音量。

**和聲的 Panning**：將和聲均勻分布在左右聲道，讓主唱獨佔正中央的空間。例如：第一和聲 L40，第二和聲 R40，第三和聲 L80，第四和聲 R80。這樣可以創造出寬闊的立體聲背景，同時保持主唱的中心地位。

## 實戰逐步流程：從乾音到成品

以下是一個完整的 Mandopop 人聲處理實戰流程，適合在任意的 DAW 中操作：

**步驟一：前置準備**

1. 將原始錄音進行音量 normalize 到 -6 dB（避免 clipping）
2. 手動剪掉不必要的空白、呼吸聲（保留自然呼吸，但去除過大或突兀的呼吸）
3. 對音準進行調校——使用 Melodyne 或 Auto-Tune，只修正明顯走音，不需做到完全的「機器感」
4. 對節奏進行 time alignment，確保人聲對齊節拍

**步驟二：初始化 EQ**

1. 掛上第一個 EQ（如 FabFilter Pro-Q 3）
2. 設定 80 Hz 高通濾波
3. 掃描 200-500 Hz 區域，衰減任何「悶」或「糊」的頻率 2-3 dB
4. 在 4 kHz 左右提升 2-3 dB（寬 Q 值）

**步驟三：壓縮器設定**

1. 掛上壓縮器（如 CLA-2A 或 RCompressor）
2. 設定 Ratio 2.5:1，Attack 20 ms，Release 50 ms
3. 調整 Threshold 至 Gain Reduction 在 4-6 dB
4. 補償 Make-up Gain 使音量匹配

**步驟四：去齒音**

1. 在壓縮器之後掛上 De-esser（如 FabFilter Pro-DS）
2. 選擇 Split-band 模式
3. 將偵測頻率設在 7 kHz，Threshold 調整至衰減量約 4 dB
4. 用 Solo 監聽齒音的處理效果，確保只處理齒音不傷其他頻段

**步驟五：空間效果**

1. 建立 Plate Reverb 輔助軌：Decay 1.5s，Pre-delay 30ms，Send 量約 -12 dB
2. 建立 Slap Delay 輔助軌：Time 80ms，Feedback 1，Send 量約 -15 dB
3. 根據歌曲段落調整 Send 量（副歌多一點，主歌少一點）

**步驟六：和聲與雙軌**

1. 如有真實雙軌，將其分別 Pan 到 L30 和 R30，音量比主唱低 8 dB
2. 將和聲軌道 Pan 到兩側，EQ 削去低頻（300 Hz 以下），適度壓縮
3. 對和聲團加入一個獨立的 Hall Reverb（Decay 2.0s），讓和聲處於主唱之後的空間位置

**步驟七：細部 Automation**

1. 對主唱做音量 automation：副歌提高 1-2 dB，主歌維持或略降
2. 對副歌的 Reverb Send 做 automation：副歌增加 Send 量
3. 對特定的高音長音或強調字做 automation 微調

**步驟八：參考比對**

1. 找一首參考歌曲（如林俊傑、鄧紫棋、周杰倫的歌曲）
2. 用耳朵比對自己的人聲處理，特別注意：亮度夠不夠？齒音刺不刺？人聲是「融入」還是「浮在」編曲之上？
3. 必要時回到前述步驟進行微調
